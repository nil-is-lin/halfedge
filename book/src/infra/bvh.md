# BVH 加速结构

> 设计文档：[bvh.pdf](../../../docs/bvh.pdf) | 源码：[bvh.rs](../../../src/bvh.rs)

## 简介

基于 AABB 的二叉树包围盒层次结构（BVH），用于加速射线求交与最近点查询等空间检索操作。

## 关键 API

| 名称 | 功能 |
|------|------|
| `Bvh` | BVH 加速结构 |
| `LEAF_MAX_FACES` | 叶节点最大面数阈值 |
